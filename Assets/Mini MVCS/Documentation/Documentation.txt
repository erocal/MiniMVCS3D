Getting Started
=============
1. Open Unity Menu: File -> Build Settings
2. Add "1. Loader" and "2. Main" to Build Settings
3. Open Unity Menu: File -> Open Scene
4. Open "1. Loader"
5. Open Unity Menu: Edit -> Play
6. Enjoy
